## 📜 Core product principles — never violate

1. **Safe Harbor** — Children only earn points, never lose them.
2. **Dopamine Bridge** — Small, frequent rewards beat large distant ones.
3. **70% Goal** — Realistic, not perfect.
4. **Partnership, not Policing** — App manages the routine, parent coaches.
5. **Planner vs Executor** — Parent plans, child executes.
